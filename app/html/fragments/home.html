Setup details

I’m running a Raspberry Pi 5 with a 16 GB SD card, which I use as a web server. On the Raspberry Pi, I’ve disabled the root user and enabled SSH access using an SSH key. File and directory permissions are configured with read/write access only where necessary, and only essential ports are open on the router. The server hosts a website running on Nginx, which is protected behind a Caddy reverse proxy. Caddy also manages my SSL certificates and enforces security and privacy headers. I’ve set up a Docker environment with multiple containers connected through a shared Docker network, exposing only a single external port. The containers are managed using Docker Compose.
