Setup details

I run a Raspberry Pi 5 as a web server. Root login is disabled and SSH access is key-based. Only required ports are open on the router and file permissions are restricted to least privilege. The site is built as a single-page application (SPA) and runs behind Nginx with a Caddy reverse proxy handling TLS, redirect rules and security headers. Services run in multiple Docker containers connected through a shared Docker network, with only one external port exposed. Containers are orchestrated via docker-compose. The system uses a PostgreSQL database.
